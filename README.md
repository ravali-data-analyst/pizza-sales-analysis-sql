🍕 Pizza Sales Analysis Using SQL
📌 Project Overview

This project analyzes Pizza Hut sales data using SQL to uncover valuable business insights from customer orders, product performance, revenue trends, and ordering behavior. The analysis demonstrates how SQL can be used to transform raw transactional data into actionable insights for business decision-making.

🎯 Business Objective
The primary objectives of this project are to:
* Analyze overall sales performance
* Understand customer ordering behavior
* Identify top and low-performing pizzas
* Calculate revenue and sales contribution
* Support business decisions using SQL
  
🛠️ Tools & Technologies
* SQL (MySQL)
* Excel / CSV
* PowerPoint
* GitHub
  
📂 Dataset
The dataset consists of four relational tables:
| Table         | Description                 |
| ------------- | --------------------------- |
| Orders        | Order date and time         |
| Order_Details | Pizza orders and quantities |
| Pizzas        | Pizza size and price        |
| Pizza_Types   | Pizza name and category     |

🗂 Database Schema
Orders
↓
Order_Details
↓
Pizzas
↓
Pizza_Types

📈 Business Questions Solved
* Retrieve the total number of orders.
* Calculate total revenue.
* Identify the highest-priced pizza.
* Find the most common pizza size.
* List the top 5 most ordered pizzas.
* Calculate quantity sold by pizza category.
* Analyze hourly order distribution.
* Count pizza types by category.
* Calculate average pizzas ordered per day.
* Find the top 3 pizzas by revenue.
* Calculate revenue contribution by category.
* Analyze cumulative revenue.
* Identify the top 3 revenue-generating pizzas within each category.
  
📌 Key Insights
* Total Orders: 21,350
* Total Revenue: $817,860.05
* The Greek Pizza is the highest-priced pizza.
* Large (L) pizzas are the most frequently ordered size.
* Classic pizzas generated the highest sales volume.
* Peak ordering hours were 12 PM–1 PM and 5 PM–7 PM
* Average pizzas ordered per day: 138.47
* Classic pizzas contributed 26.91% of total revenue.
* Revenue showed consistent growth throughout the analysis period.
